Gigatrees is a free, Windows 64 utility written by Tim Forsythe (© 2016).

It can be used to validate a GEDCOM file, generate standalone genealogy related reports, or build an entire family tree website.

Gigatrees provides many unique features, including support for the Genealogy Proof Standard, data consistency checking, census table reports, population distribution heatmaps, location maps, generation lists, kinship lines, full language translations and more.
